# mongodb-project

Instructions for cloning the project 
  - Open Terminal 
  - cd vagrant 
  - cd vagrant 
  - git clone *github link* 
  
  Installations 
  - cd *folder name* 
  - vagrant ssh 
  - install npm init -y 
  - install npm i 
  
  Config.json 
  - Create config.json in root folder 
  - Add...
{
"MONGO_USER" : "*username*",
"MONGO_PASSWORD : "*password*", 
"MONGO_CLUSTER_NAME : "*clustername*" 
}
  
  Running Server
  - vagrant ssh 
  - nodemon -L index.js (-L not needed from home computer)
